액트,레벨 별 체크 포인트

  • 1장 [ 빨2파1(토템-깊집-화피추) / 파2(화질-비쇄) / 빨1초1(강타-맹보) ]
    • 1렙: 보상 강타 / 동결파동 구입
    • 4렙: 보상 비전쇄도,신성한 화염토템,서리 점멸 / 깊어지는 집중 유지 구입(맹공은 선택)
    • 10렙: 보상 화염질주,화염피해추가or번개피해추가 / 미끼토템 구입(선택)
    • 12렙: 폭풍점사
  • 2장
    • 16렙: 보상 재의 전령(불토템 쓰는 동안만) / 폭풍의 방패, 신념의 파도 구입
    • 18렙: 보상 원소 집중 / 제어된 파괴, 물리 번개 전환 구입
  • 3장
    • 24렙: 보상 전도성 / 쇠약화 구입
    • 33렙: 전직
  • 4장
    • 마지막: 보상 다중 토템
  • 기타사항
    • 55레벨: 2차 전직
    • 68레벨: 3차 전직
    • 75레벨: 4차 전직
    • 목걸이 성유: 철목(투명,호박,청록)
    • 플라스크
      • 생명력(신성한, 60랩제)
        접두 - 즉시회복,반반회복
        접미 - 출혈제거
      • 수은, 다이아, 석영
        접두 - 충전 소모량 감소
        접미 - 시전속도 증가, 화상이나 중독제거등
      • 하이브리드or마나
        접두 - 즉시회복,반반회복
        접미 - 화상이나 중독제거등
