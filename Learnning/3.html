JavaScript

'JavaScript'란?


- 사용자와 동적으로 상호작용 가능한 기능의 필요성에 의해 등장
- HTML을 제어하는 언어

실습에 사용된 'HTML' 태그

  • input
  • script: 'Javascript'를 사용하는 태그

실습에 사용된 ('Javascript'를 사용하는)'HTML'의 속성

  • onclick
  • onchange
  • onkeydown

실습에 사용된 'css' 효과(선언-declaration)

  • background-color
  • font-weight:bold

실습에 사용된 'Javascript'의 속성

  • alert
  • document.write
  • if(true): 조건문
  • document.querySelector('body')
  • ["~","~"]: 배열. 불러올때 첫번째 데이터의 호출값은 0부터
  • push: 배열에 데이터 추가
  • while(true){}: 반복문
  • function ~(): 함수를 사용하는 명령어
  • 함수의 입력
    • 매개변수(parameter): 함수의 ()안에 들어가는 변수 지정자 정의(self)(color)
    • 인자(arguments): 함수의 ()안에 들어가는 실제 변수값(this)(실제적용할색)
  • 함수의 출력: 함수의 {}안에 return
  • for(var key in ~){} : 객체의 이름값들을 하나하나 반복해서 {}를 실행한다.

실습에 사용된 'jQuery'

  • $(): jQuery를 사용하는 명령


기타 정리

- 큰따옴표안의 문자열에 변수를 넣기 위해서는 "+name+"라는 형태를 사용한다.

- 변수를 사용할때는 var로 표기, 중복되는 코드를 제거하기위해 사용

- '===' 비교 연산자, 이항 연산자

- < less than, 즉

- true와 false를 묶어서 'boolean' 이라고 한다.

- document.querySelector('')가 선택하는 대상이 자기 자신일 경우 this로 대체가능하다.

- 코드의 중복을 제거하기 위해 정리하는 작업을 리팩토링이라고 하며 리팩토링 과정에서는 함수가 중히 쓰인다.

- 수납장 같은 역할을 하는것 배열=array(배열은 정보가 순서대로 저장), 함수

- 반복문 사용시 제어에 필요한 변수의 이름값은 관습적으로 'i'를 사용한다.

- 객체의 다면적인 성격중 하나는 서로 연관된 함수들을 정리 정돈하는 수납장, 폴더 디렉토리와 같은 개념

- 객체에 속해있는 함수는 메소드라고 부르고 변수는 프로퍼티라고 부른다.

- 배열은 ['~'], 객체는 {"이름":"~"}

- 객체에 담을 데이터는 반드시 이름을 필요로 한다.

- 객체를 출력할때는 객체.이름

- 객체를 출력할때 이름에 띄어쓰기가 있으면 . 이 아니라 객체["~"]

- Library, framework: 남이 만들어 놓은 코드를 내가 활용하는 방식

- jQuery: 대표적인 Library
