AutoHotKey

명령어 목록

  • 변수이름 :=값 : 변수를 생성하고 값을 저장
  • 변수이름 = 값 : 변수에 저장된 값과 특정 값을 비교
  • 변수이름 = 변수이름 : 각 변수의 값을 비교
  • %변수이름% : 변수에 저장된 값을 명령어에 사용
  • Gui, Add, Gui요소(text,edit등등), 좌표 v변수이름, 표기택스트 : gui변수 설정법
  • Guiconrtrol, ,변수이름,바꿀내용 : Gui 변수를 제어
  • Gui,submit,nohide : Gui 변수 업데이트
  • !:Alt / ^:Ctrl / +:Shift
  • gosub: 타겟라벨이 위치한곳으로 이동했다가 return을 만나면 돌아옴
